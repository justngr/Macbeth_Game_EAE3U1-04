# Macbeth_Game_EAE3U1-04
Macbeth Project "Pick Your Poison"

Attached is the download file for my Macbeth project (Pick Your Poison). For this project I’ve decided to make a video game that takes you through a summary of every act and makes you complete a couple multiple answer questions. There is also an extra game mode that tests your skill by making you complete multiple choice questions. My game is on Github as classroom would not let me submit it. 

Please note that .exe files cannot be opened with a macbook and should be opened with a windows based laptop. HOWEVER with the use of applications like VM Fusion or Whisky on a Macbook lets you run it, but it may not work well. 

After downloading the game and booting it up for the first time you may be warned of a virus or malicious software. This is completely normal as the file was converted from a python script to .exe. If you click “More Info” you will be able to run the application anyway. If you have a problem with this you can always ask to play it on my computer as it does seem risky.
